Attribute rules

This rule is based on using tags and names of the element properly because you are going to nest them in your document.

  1. There must be [Attribute name = attribute value], in your value must be there is quot.

  2. Attribute are separated with spaces but in value you should use underscore_

  3. Attribute name must be lowercase.

  4. Element type have private attributes.

  5. Public attribute can go on any element.
    - Classify (class): categorising
    - identity (id): identification

  6. Do not use & n b s p to make space but use <br> or <p> instead.

Definition

  1. HTML:

    HTML stands for Hypper Text Markup Langage.

    HTML is the language used to create webpages.
    Hypertext refers to the hyperlinks that an HTML page may contain.
    Markup language refers to the way tags are used to define the page layout and elements within the page.

  2. Elements:

    In general
    Element is a part or feature of a whole systeme, plan, piece of work which is very important. It can be a group of something such as people, animals, things.

    In HTML
    Element usually consists with a start tag and end tag, with content inserted in between.

List of HTML rules

Definiton

Rule is an official instruction that says how things must be done, need to be done or what is allowed, especially in a game, organisation or job as well.

  1. Before getting started to code, you have to create a file which must be lowercase , as well as your folder.

  2. Define a <!DOCTYPE.html> at the top of the document.

  3. Every start tag, do not forget to start with ( < ) and to end with ( > ).
    The same in ending tag, open with( </ ) and close with( > ).

  4. Element: Usualy consists of a start tag and an end tag, with the content inserted in between. Make sure that you know the meaning of the element which you use, like strong and bold tags.

  5. Headings: Your <h1> - <h6> should be appeared to make title or subtitle only. Don't use it for Bold, Big and Small letter.

  6. Paragraph: Closing </p> tag mustn't be forgotten. Never make <p> descendent of other <p> tag.

  7. Comment: It is good to use a comment in a document but remember that it will not apper in the browser.

  8. Link: Not only URL can be in the link but also it can be image, but make sure that you use the apropriate link.
    Without a forward slash at the end of subfolder addresses, you might generate two requests to the server.
    Many servers will automatically add a forward slash to the end of the address, and then create a new request.
    Do not forget to use ....

  9. Section: Do not use section as like an element that can be used whatever the descendent, but use it for sectioning document.
    As an example, a navigation menu should be wrapped in a <nav> element, but a list of search results and a map display and its controls don't have specific elements, and could be put inside a section.

  10. Form: Every time you create an html form, you must start it by using <html> element, then nesting all content inside.

  11. Inline markup: It's a good idea to tell your computer how to read by using inline, but make sure that you nest it properely, like using <button> and <kbd>, that you may get errors.
